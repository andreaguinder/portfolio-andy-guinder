Administrador de citas

[ Ver proyecto ]

La interfaz presenta un diseño funcional dividido en dos bloques principales. En el primero, se encuentra un formulario detallado permite ingresar los datos del dueño y la mascota, incluyendo la selección de síntomas, fecha y horario. Al presionar el botón de creación, la cita se genera dinámicamente en el segundo bloque, donde los turnos se organizan verticalmente separados por divisores claros. Cada tarjeta de cita incluye opciones para eliminar o editar; esta última recarga la información en el bloque del formulario para su modificación, permitiendo una gestión ágil y organizada. Reformado en 2025.

Cotizador de seguros

[ Ver proyecto ]

La aplicación presenta un diseño web responsive optimizado para una navegación fluida en una sola columna. La interfaz cuenta con selectores intuitivos para definir la marca y el año del vehículo, además de una opción para elegir entre una cobertura básica o completa. Al accionar el botón de cotización, se activa un spinner animado que mejora la experiencia de espera del usuario mientras se procesan los datos. Finalmente, el resultado se despliega debajo, mostrando un resumen detallado de la configuración elegida junto al monto final del seguro, logrando un proceso de consulta rápido y visualmente atractivo. Reformado en 2025.

Formulario

[ Ver proyecto ]

Este formulario cuenta con un sistema de validación avanzada mediante funciones de JavaScript y expresiones regulares (Regex), asegurando que cada campo cumpla con formatos específicos de contacto y longitud mínima. La interfaz ofrece feedback en tiempo real con mensajes de error individuales por cada ítem y una alerta general ante datos incompletos. Incluye funciones para restablecer los campos o enviarlos, acción que dispara un modal de confirmación tras validar el éxito del proceso. Además, el diseño es totalmente web responsive, garantizando una experiencia óptima en cualquier dispositivo. Reformado en 2025.

Buscador multipes campos

La interfaz de usuario para el buscador es intuitiva. En la parte superior, encontrarás un conjunto de menús desplegables que te permiten seleccionar los diferentes filtros, como marca, año y color.
